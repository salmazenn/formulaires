Configurer EmailJS en 5 min

  1. Creez un compte gratuit sur emailjs.com (200 emails/mois offerts)
  2. Connectez Gmail : Email Services > Add New Service > Gmail > Connect Account. Notez votre Service ID
  3. Creez un template : Email Templates > Create New > mettez {{message}} dans le corps > Save. Notez votre Template ID
  4. Copiez votre Public Key : Account > General > Public Key
  5. Collez ces 3 valeurs dans la configuration du formulaire avec votre email de reception
